FastZapret — удобный лаунчер с веб-панелью поверх проверенного движка winws. Discord, YouTube, Telegram и онлайн Le Mans Ultimate. Запустил — и работает.

Скачать Исходники на GitHub

Что внутри

Ядро обхода — winws из проекта bol-van/zapret, годами доведённый до ума. FastZapret делает его удобным.

Проверенный движок

Под капотом — winws и стратегии из проекта Flowseal. То, что реально работает у тысяч людей, а не самоделка.

Веб-панель

Локальная страница на 127.0.0.1:7890: включение, выбор стратегии, тумблер Le Mans Ultimate и логи движка.

Профиль Le Mans Ultimate

Отдельная мягкая стратегия под онлайн LMU (Cloudflare + Hetzner). Не ломает игровую сессию — не выкидывает в лобби.

Простые .bat

Не любишь интерфейсы? Правый клик по general.bat или Le Mans Ultimate.bat → запуск от админа. Всё.

Несколько стратегий

default / alt / alt2 / alt3. Не пробивает у твоего провайдера — переключил вариант в один клик.

Не VPN

Трафик не шифруется и не гоняется через чужой сервер — только правится обход DPI. Быстро и без прокси.

Le Mans Ultimate

Онлайн LMU в РФ ломается, потому что игра ходит в две разные сети. FastZapret обрабатывает обе.

Часть игрыГде хоститсяЧто делает профиль LMU
API / логин / лоббиCloudflareобычный TLS-desync — пробивает надёжно
гоночные сервераHetznerмягкий desync по первым пакетам
игровая сессия—не трогается (cutoff) → не кикает
⚠️ Честно: часть серверов на Hetzner провайдеры режут жёстче, чем берёт чистый DPI-обход. Если после включения профиля какие-то сервера всё равно не пускают — запустите вместе с FastZapret бесплатный Cloudflare WARP.

Установка за 30 секунд

Ни .NET, ни VPN, ни настроек.

Скачать

Кнопка «Скачать» сверху, либо Code → Download ZIP на GitHub. Распаковать в любую папку.

Запустить от админа

Правый клик по general.bat (или Le Mans Ultimate.bat) → «Запуск от имени администратора». Или fastzapret-gui.exe для панели.

Играть

Окно оставить открытым. Discord, YouTube и онлайн LMU уже работают.
